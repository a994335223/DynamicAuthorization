#### DynamicAuthorization
Android 动态授权
```html
<Step 1. Add it in your root build.gradle at the end of repositories>
		allprojects {
			repositories {
			...
			maven { url 'https://jitpack.io' }
			}
		}



<Step 2. Add the dependency>
		dependencies {
	  		implementation 'com.github.a994335223:DynamicAuthorization:v1.0'
		}
```
